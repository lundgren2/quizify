# Socify

### A project for devX 2018
- 24h hackathon
- Live preview: [https://socify-app.herokuapp.com](https://socify-app.herokuapp.com)

## Setup

**Dependencies:**

- Yarn
- Node.js

**Tools**
- React.js
- React Router
- Spotify SDK
- emotion

### Client

1. `cd client && yarn`
2. run `yarn start`

### Server

1. `cd server && yarn`
2. run `node server.js`
