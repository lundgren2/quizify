- 24h hackathon
- Live preview: [https://socify-app.herokuapp.com](https://socify-app.herokuapp.com)

## Setup

**Dependencies:**

- Yarn
- Node.js

**Tools**

- React.js
- React Router
- Spotify SDK
- emotion

## Instructions

1. `$ git clone` this repo
2. `$ yarn` to install all dependencies
3. `$ yarn start` to run the server
